Dankbaarheidsdagboek

Deze tool helpt je om dankbaarheid een onderdeel te maken van je dagelijks leven.

Wat heb je aan deze tool?

Dankbaarheid heeft een enorm positieve invloed op je geluksgevoel en algehele gezondheid.

Deze tool helpt je om dankbaarheid een onderdeel te maken van je dagelijks leven.

Hoe werkt het?

  • Installeer de tool in je ontwikkelomgeving
  • Je vind de tool in het prive gedeelte van je ontwikkelomgeving onder Tools
  • Beschrijf zo vaak je wilt waar je dankbaar voor bent
  • Je ontvangt wekelijks een email-notificatie ter herinnering
